# NovaTide Logistics Enterprise Profile

Welcome to the canonical enterprise profile for **NovaTide Logistics**, a fictional multinational logistics and supply chain organization used throughout the Governance Lab project.

Rather than creating a new fictional organization for each repository, all Governance Lab projects reference the same enterprise context. This enables realistic, interconnected demonstrations of governance, risk, compliance, cybersecurity, AI governance, and third-party risk management.
